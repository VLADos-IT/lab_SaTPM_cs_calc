# cs_minimal_calc

## Цель

Реализовать ПО, имеющее минимальный функционал калькулятора для выполнения основных и дополнительных арифметических операций с двумя числами.

## Задачи

* Основные арифметические операции работают для двух заданных чисел: сложение (+), вычитание (−), деление (/), умножение (*)
* Дополнительные операции для двух заданных чисел: возведение в степень (^), вычисление корня (sqrt)
* Обеспечить получение округленного значения результата вычисления до 3 знаков после запятой

**Исполнитель:** Лутков В.

**Заказчик:** = <>

---

## Алгоритм работы ПО

1. Пользователь вводит первое число для выполнения операции
2. Пользователь вводит символ операции (`+`, `-`, `*`, `/`, `^`, `sqrt`)
3. Пользователь вводит второе число (если требуется; для `sqrt` второе число отсутствует)
4. Пользователь получает ответ в выводе в консоль
5. Программа завершает работу

---

## Требования к реализации ПО

### 1. Обработка ошибок

* Пользователь должен вводить значения чисел с точностью не более 5 знаков после запятой (0.00000)
* Нельзя выполнять операцию деления на 0
* Пользователь не может вводить некорректные типы данных (должны быть числа)
* При вводе некорректного символа операции программа сообщает об ошибке и предлагает повторить ввод
* При вводе отрицательного числа для `sqrt` программа выводит:
  **«Невозможно вычислить квадратный корень из отрицательного числа»**

### 2. Корректные вычисления

* Округление результата до 3 знаков после запятой (0.000)
* Математически корректное округление

### 3. Синтаксическая реализация

* Соответствие синтаксису выбранного языка программирования
* Понятный и читаемый интерфейс консоли пользователя

## Некорректные вводы и ошибки

### Некорректный ввод числа

```text
Ввод:
Введите первое число: АБВ

Вывод:
Данное значение не является корректным для ввода. Введите верное число.
Введите первое число: _
```

### Неизвестная операция

```text
Ввод:
Введите первое число: 10
Введите операцию(+,-, *, /, ^, sqrt): #

Вывод:
Введённая операция не поддерживается. Повторите ввод.
Введите операцию(+,-, *, /, ^, sqrt): _
```

### Превышено количество знаков после запятой

```text
Ввод:
Введите первое число: 1.234567

Вывод:
Превышено количество знаков после запятой (максимум 5).
Введите первое число: _
```

### Корень из отрицательного числа

```text
Ввод:
Введите первое число: -9
Введите операцию(+,-, *, /, ^, sqrt): sqrt

Вывод:
Ошибка: Невозможно вычислить квадратный корень из отрицательного числа.
Введите первое число: _
```

### Деление на ноль

```text
Ввод:
Введите первое число: 10
Введите операцию(+,-, *, /, ^, sqrt): /
Введите второе число: 0

Вывод:
Ошибка: Деление на 0 невозможно. Введите корректное значение.
Введите первое число: _
```
